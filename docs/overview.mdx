### What is it?

Amethyst is a [music player](https://en.wikipedia.org/wiki/Media_player_software), **it is NOT a streaming service**

### Who made it?

Amethyst was started by [Geoxor](https://www.youtube.com/@Geoxor) and now is a big [open source project](https://github.com/Geoxor/amethyst) supported by multiple developers

### Why did you make it?

I was looking for a sideproject that was fun and took my mind off drama and stress, I also wanted an app that would render [spectrums](/user-manual/visuals/spectrum-analyzer) and [vectorscopes](/user-manual/visuals/vectorscope) at native refresh rate because [VST plugins](https://en.wikipedia.org/wiki/Virtual_Studio_Technology) are slow

### Is it malware?

No, **Amethyst is not malware**, it simply is a [music player](https://en.wikipedia.org/wiki/Media_player_software) app that plays audio files and has some online integrations that are customizable

### Does it collect my data?

No, Amethyst does not have a web server that stores any user data, the only data it can send out online is currently the [last.fm scrobbling](https://www.last.fm/api/scrobbling) integration
