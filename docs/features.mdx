Amethyst has many features that work on all platforms

### Features Table

| **Feature**                                                       | **Description**                                               |
| ----------------------------------------------------------------- | ------------------------------------------------------------- |
| 🎮 Discord Rich Presence                                         | Show what you're playing in real time on your [Discord status](https://support.discord.com/hc/en-us/articles/360040816151-Rich-Presence) |
| 🎵 Last.fm Scrobbling                                            | Automatically log tracks to your [Last.fm account](https://www.last.fm/about/trackmymusic) |
| 🏷️ ID3 Metadata Viewer                                           | View [audio file metadata](https://en.wikipedia.org/wiki/ID3) with ease |
| 📂 Folder Import with Directory Monitoring                       | Watch folders for changes and auto-import new tracks |
| 🎨 Customizable Themes                                           | Personalize the [interface](https://en.wikipedia.org/wiki/Graphical_user_interface) to match your vibe |
| 🎧 ASIO/CoreAudio Support                                        | Low-latency audio playback with pro-grade drivers |
| 🧠 Built-in DSP                                                  | [Real-time audio effects](https://en.wikipedia.org/wiki/Digital_signal_processing) and processing |
| 🔊 Output Diagram                                                | Visualize [signal flow](https://en.wikipedia.org/wiki/Audio_signal_flow) for easier routing |
| 🌐 Local Discovery Algorithm                                     | Auto-detect and index local audio content intelligently |
| 🎚️ Multichannel Audio                                            | Play [multichannel audio](https://en.wikipedia.org/wiki/Multichannel_audio) with full control |
| ⌨️ [Keyboard Shortcuts](/user-manual/keyboard-shortcuts)         | Navigate quickly and boost productivity with [hotkeys](https://en.wikipedia.org/wiki/Keyboard_shortcut) |
| 🧭 Command Palette                                               | Access all [commands](https://code.visualstudio.com/docs/getstarted/userinterface#_command-palette) with a quick searchable launcher |
| 📏 LUF Meter                                                     | Accurate loudness measurement in LUFS |
| 📈 [Vectorscope Analyzer](/user-manual/visuals/vectorscope)      | Visualize [stereo image](https://en.wikipedia.org/wiki/Stereophonic_sound) and [phase correlation](https://en.wikipedia.org/wiki/Phase_correlation) |
| 📊 [Spectrum Analyzer](/user-manual/visuals/spectrum-analyzer)   | Monitor [frequency content](https://en.wikipedia.org/wiki/Spectrum_analyzer) in real time |
| 🌈 Spectrogram Analyzer                                          | Visualize audio over time across the [frequency spectrum](https://en.wikipedia.org/wiki/Spectrogram) |
| 🔄 Oscilloscope Analyzer                                         | Get a [waveform view](https://en.wikipedia.org/wiki/Oscilloscope) of your audio signal |
