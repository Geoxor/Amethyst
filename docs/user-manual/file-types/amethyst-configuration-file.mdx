### What is it?

Amethyst allows for exporting user settings as a `.acf` [JSON](https://en.wikipedia.org/wiki/JSON) file for backup or for sharing settings to other users

<Tip>.acf files can be edited in text editors, this might be useful for fixing issues that cause Amethyst to not run properly</Tip>
